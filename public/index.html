Parts Endpoints

GET /parts

Returns all parts as { items: [...] }.

GET /parts/:id

Returns a single part by _id.

POST /parts

Creates a new part. Validated with Joi: { name, brand, category, image, price }.

Builds Endpoints

GET /builds

Returns all builds as an array.

GET /builds/:id

Returns a single build by id.

POST /builds

Creates a new user submission. Validated with Joi: { car, instagram, mods, image? }.

Dev Utility

GET /dev/reset-builds

Removes user-submitted builds (ids starting with user-) and keeps seeded builds only.
